### 🚀 Overview
**Gift Store** is a modern, high-end storefront built to showcase curated tech discovery. It prioritizes a premium user experience with a focus on intent-based shopping and sleek visual feedback.

### ✨ Key Features
- **Dynamic Home Experience**: Features a high-conversion hero section, category-based discovery (Students, Creatives, Gaming), and trending product grids.
- **Instant UI Feedback**: Real-time cart synchronization. Adding or removing items updates the header count immediately.
- **Trust-First Design**: Integrated highlights for Secure Shopping, Premium Warranties, and Express Shipping.
- **Modern Routing**: Clean page transitions powered by React Router 7.
